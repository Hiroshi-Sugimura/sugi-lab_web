Serial ESP Wi‑Fi Configurator (Web Serial)


  


  

    * Web Serial API は Chromium 系（Chrome/Edge）でご利用ください。

  


  

  

    
■ このウェブサイトについて

    

      Wi‑Fiの設定を、再コンパイルせずに書き換えができるツールです。
    


    
■ 使い方

    

      ポートを選択
Wi‑Fiの SSID / Password を入力
設定送信で書き込み
Wi-Fi設定の初期化でリセット
※ ポート更新で見つからない場合は、特定のドライバーが必要です。
